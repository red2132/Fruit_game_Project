# 과일 충돌 게임

## 1. 개발 목표
Nods.js의 패키지인 Matter-js를 이용해 간단한 게임 제작
- keyevent를 이용한 과일 이동
-collision  기능을 이용해, 같은 과일끼리 부딪치면 더 큰 과일로 합쳐지는 
  기능 제작,
- 기준 선을 넘으면, GAME OVER 되는 기능 제작
